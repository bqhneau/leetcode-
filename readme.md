## 1、时间复杂度
### O(1) --顺序执行的代码
```js
    let i = 0;
    i++;
```
    
### O(n) --for循环
```js
    for(i=0;i++;i<10) {
        i++
    }
```

### O(1) + O(n) = O(n)
```js
    let i = 0;
    i++;
    for(i=0;i++;i<10) {
        i++
    }
```

### O(n) * O(n) = O(n^2)
```js
    for(i=0;i++;i<10) {
        i++
        for(j=0;j++;j<10){
            j++
        }
    }
```

### O(logN) --while循环
```js
    while(i<n){
        i = i * 2
    }
```

## 2、空间复杂度
### 2.1 空间复杂度是什么？
    算法在运行过程中临时占用存储空间的大小

### 2.2 O(1) --单个变量/简单变量
```js
    let i = 0;
    i++
```

### 2.3 O(n) --一维数组
```js
    let arr = [];
    for(i=0;i++;i<10){
        arr[i] = i
    }
```

### 2.4 O(n^2) --二维数组/矩阵
```js
    const arr = [];
    for(i=0;i++;i<10){
        arr.push([]);
        for(j=0;j++;j<10){
            arr[i].push(j)
        }
    }
```
