## 1、空间复杂度是什么？
    算法在运行过程中临时占用存储空间的大小

## 2、O(1) --单个变量/简单变量
```js
    let i = 0;
    i++
```

## 3、O(n) --一维数组
```js
    let arr = [];
    for(i=0;i++;i<10){
        arr[i] = i
    }
```

## 4、O(n^2) --二维数组/矩阵
```js
    const arr = [];
    for(i=0;i++;i<10){
        arr.push([]);
        for(j=0;j++;j<10){
            arr[i].push(j)
        }
    }
```
